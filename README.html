This collection of php scripts is designed to be used for automated searching and downloading (using hellanzb) of new TV Shows from usenet on a daily basis, based on information from http://on-my.tv/, http://pogdesign.co.uk/cat/ or http://tvrage.com/.

Once shows have been downloaded they are then sorted into either pre-specified directories, or into an "unwatched" directory to allow the user to see what shows they havn't yet watched. It is also possible for the scripts to automatically move shows from the "unwatched" directory into a "watched" directory once the show has been watched. (requires inotify)

Currently due to limitations imposed by the terms and conditions of newzbin.com, the actual code to search newzbin can not be released.

requirements

These scripts require php5-cli, and the pecl package inotify
These can be installed (on ubuntu) by using: 

sudo apt-get install php5-cli php5-dev
sudo pecl install inotify-`pecl remote-info inotify | grep ^Latest | head -n 1 | awk '{print $2}'`
sudo su -c "echo 'extension=inotify.so' >> /etc/php5/cli/php.ini"

using the code

Once the code has been downloaded, edit config.php to your liking and then run daemon.php to handle the automated functions.

The contents of the www directory should be made available as specified by $config['tv']['urlbase']. The directory structure and layout must remain the same (so no moving the contents into /var/www/ unless you want to move the other scripts into /var) otherwise the scripts will be unable to find each other.
